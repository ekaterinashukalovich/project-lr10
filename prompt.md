Данный файл содержит точные промпты, использованные при генерации и доработке фронтенд-части (static/) сервиса ипотечного скоринга.

\## Промпты по файлам

\### 1. static/index.html

**Промпт:**

Создай семантическую HTML-структуру для веб-приложения ипотечного скоринга. Включи:

- Форму ввода данных клиента с полями: age, gender, education, home_ownership, loan_amnt, int_rate, percent_income, cred_hist_length, credit_score, previous_defaults
- Секцию для загрузки CSV-файла
- Секцию для загрузки модели (.pkl) для администраторов.
- Контейнеры для вывода результатов предсказаний.

\### 2. static/style.css

**Промпт:**

Напиши современный адаптивный CSS для формы ипотечного скоринга. Используй белый фон с легкой тенью

\### 3. static/app.js

**Промпт:**

Напиши vanilla JavaScript для взаимодействия с FastAPI backend (/predict, /predict-from-csv, /upload-model). Нужно реализовать:

-  Сбор данных из формы в массив объектов JSON. Валидация перед отправкой.
- Вероятность одобрения в %, метка (approved/rejected), ROC-AUC.
- Индикация загрузки: блокировка кнопок, смена текста на "⏳ Обработка...", визуальная обратная связь.
- Обработка ошибок API (400, 422, 500) с выводом сообщений.
